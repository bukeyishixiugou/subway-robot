# PySide6 DLL 加载失败 - 解决方案

## 问题描述
```
ImportError: DLL load failed while importing QtWidgets: 找不到指定的程序。
```

## 解决方案（按优先级排序）

### 方案 1: 安装 Visual C++ Redistributable（最可能有效）

1. 下载 Visual C++ Redistributable:
   - 64位: https://aka.ms/vs/17/release/vc_redist.x64.exe
   - 32位: https://aka.ms/vs/17/release/vc_redist.x86.exe

2. 安装后重启终端

3. 重新运行程序:
   ```powershell
   python main.py
   ```

### 方案 2: 使用 PyQt6 替代（推荐）

PyQt6 和 PySide6 API 几乎完全相同，但 PyQt6 在 Windows 上通常更稳定。

```powershell
# 卸载 PySide6
pip uninstall PySide6 -y

# 安装 PyQt6
pip install PyQt6

# 运行替代版本
python main_pyqt6.py
```

### 方案 3: 重新安装 PySide6

```powershell
# 完全卸载
pip uninstall PySide6 -y

# 清理缓存
pip cache purge

# 重新安装
pip install PySide6 --no-cache-dir
```

### 方案 4: 检查 Python 版本

PySide6 推荐使用 Python 3.8-3.11。如果使用 Python 3.12+，可能会有兼容性问题。

```powershell
python --version
```

如果版本过高，考虑使用 conda 创建 Python 3.11 环境:
```powershell
conda create -n py311 python=3.11
conda activate py311
pip install PySide6
```

## 快速测试

运行诊断脚本:
```powershell
python diagnose_pyside6.py
```

## 推荐操作

**最简单的方法**: 直接使用 PyQt6
```powershell
pip install PyQt6
python main_pyqt6.py
```
